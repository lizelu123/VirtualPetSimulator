Welcome to Virtual Pet Simulator.

This page contains the new Java Virtual Pet Simulator, version alpha. This simulator is unlike most Java simulators, especially because of our features. Some features include:

  • Dynamic aging
  • Real time pet alerts
  • Easy GUI and saving system
  • much more...

Downloading Information

When you download this file as a ZIP or TAR Ball, it will contain multiple files including all files required to run Virtual Pet Simulator. If you don't know what you're doing, use Install.exe. (not available now) Otherwise, set your directory to /out/artifacts/ and run App.jar.

Application Updates

This application will be updated on a daily basis, with upgrades to the program and new features as well. In a later day, the updater will be included in the program.
